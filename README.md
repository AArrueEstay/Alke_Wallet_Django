# 🏦 Alke Wallet

Aplicación web tipo billetera digital desarrollada con Django.

## 🚀 Funcionalidades

- CRUD de usuarios
- Gestión de saldo
- Depósitos y retiros
- Validación de saldo
- Email único

## 🛠️ Tecnologías

- Python
- Django
- SQLite
- Bootstrap

## ▶️ Cómo ejecutar

```bash
python manage.py runserver
